# Project Log

Shared status log for everyone's Claude/Codex sessions to read before starting work, and update after.

- `STATUS.md` — current snapshot: what's done, in progress, blocked, and next. Always overwrite this to reflect the latest state.
- `YYYY-MM-DD-<name>.md` — dated entry per work session, append-only. One file per person per day is fine; add to it if you work again that day.

Convention for dated entries:

```
## <name> - <time>
What I worked on:
What changed:
What's next / blockers:
```

Any agent (Claude, Codex, or otherwise) picking up this project should read `STATUS.md` first.
