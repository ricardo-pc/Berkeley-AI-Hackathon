# Prescription Requirement Eligibility Agent

Checks whether patient meets requirements to get a refill:
- Has a previous visit within the past 6/12 months (depends on doctor, usually based on age).
- Has an upcoming visit within the next year.
- Has been prescribed the medication before with identical dosage and instructions.
- Has not been prescribed conflicting medication (mirrors the warning MyChart/eClinicalWorks would show).
